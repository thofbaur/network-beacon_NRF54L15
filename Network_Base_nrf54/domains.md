# Architecture
This code shall contain the following domains (files):
- main
- led
- radio
- nus

# Functionality
- main shall contain basic functions for booting and intialization
- led shall contain all activities related to led.
- radio shall contain all function needed for radio initialization, starting, stopping, scanning, etc.
- nus shall contain the sending and receiving data via the NUS

All files shall expose the functions needed from other domains.
